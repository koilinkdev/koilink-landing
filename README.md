# рҹҢҹ Material UI + Next.js (App Router) вҖ” TypeScript Starter Template

This project is a modern starter template that combines Next.js App Router with Material UI (v7) and TypeScript, designed for scalable, themeable, and accessible web applications.

## рҹ”§ Key Features

- Next.js App Router (v15) with full support for server components and layouts
- Material UI v7 with modern @mui/material-nextjs integration
- TypeScript for type-safe, maintainable development
- Dynamic Theming using createTheme with support for light and dark modes
- Custom Palette abstraction for consistent color management
- Typography configured with [Google Fonts (Roboto)] via next/font
- CSS Variables enabled using MUIвҖҷs CssVarsProvider for real-time theme switching
- Responsive Layout with a hybrid component styling approach using both @mui/materialвҖҷs styled() API and SCSS


# рҹҺЁ Theme & CSS Variable Setup

- Themes are dynamically generated using a getAppTheme(mode) function that creates palette settings based on the current mode (light or dark)
- The palette logic is modularized via a getPalette(mode) utility
- CSS variables (--mui-*) are automatically generated by MUI, making the theme accessible at the CSS level for SCSS and other stylesheets
- Future-ready toggle support via MUIвҖҷs useColorScheme API
- Built-in support to later integrate full system preference detection or manual theme switching
